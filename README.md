# DeGraoEmGrao
🍎 De Grão em Grão: A specialized management platform for Food Banks designed to reduce waste, optimize logistics, and connect donors with NGOs.

### 💻 Tech Stack & Requirements
* **Language:** Java (JDK 17+)
* **Database:** SQL (MySQL 8.0 / PostgreSQL)
* **JDBC Driver:**

### 🗄️ Database Setup
1. Execute the script found in `/database/db_schema.sql` to create the tables.
2. Update the `db.properties` or connection string in Java with your local credentials.
